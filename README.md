## Asteroids Game
#### Made with Allegro game programming library.

### Controls:

- `ESC`: exit
- **Green Player:**
  - `LEFT`: turn left
  - `RIGHT`: turn right
  - `UP`: thrust
  - `DOWN`: reverse
  - `SPACE`: fire
- **Blue Player:**
  - `A`: turn left
  - `D`: turn right
  - `W`: thrust
  - `S`: reverse
  - `TAB`: fire
